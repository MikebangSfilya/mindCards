# mindCards

## Что за сервис?

Это REST API CRUD сервис для карточек памяти, содержат вопрос/ответ, либо же просто заголовок и его описание. Подходят для запоминания различной информации и интревального повторения.

Есть 5 уровней запоминание информации, как только достигается 5 карточки больше не будут появляться долго время.
